# GentleBot
- - -
## 어떻게 설치 하나요?
- 1.이 레포지토리를 포크합니다.
- 2.포크한 레포지토리를 클론합니다.
- 3.cmd에 진입해 cd [경로] 를 입력합니다.
- 4.cmd에 ```npm install```을 입력합니다.
- 5.```bot.inc.json```파일과 ```keys.inc.json```파일의 파일명을 각각 ```bot.json```과 ```keys.json```으로 바꿔줍니다.
- 6.```bot.json```파일과 ```keys.json```파일을 적절히 수정합니다.
- 7.cmd에 ```node bot```을 입력합니다.
